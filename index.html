About this repository

This is a minimal yet beautiful static site — HTML + CSS only — specifically designed to be deployed to GitHub Pages with a GitHub Actions workflow.

Perfect for learning modern development workflows while creating something visually appealing.

Branching

Create feature branches, open pull requests, and merge to main following best practices.

Workflow

GitHub Actions automatically build artifacts and deploy to GitHub Pages on pushes to main.

Protection

Use branch protection rules to require reviews and passing checks before merging.

Hands-on

Make changes on a branch, push, open a PR — then watch it deploy automatically when merged.
